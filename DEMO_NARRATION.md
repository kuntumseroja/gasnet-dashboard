## Appendix A: Executive Q&A Preparation

### Finance Questions

| Likely Question | Prepared Answer |
|----------------|-----------------|
| "How accurate is the financial data?" | All baseline figures are sourced from PGN's audited FY2024 annual report and Q1 2025 quarterly filing. The AI validates against published data with ±0.1% tolerance. |
| "Can this connect to our actual SAP?" | Yes — production architecture uses SAP BTP OData APIs. Current demo uses realistic mock data based on actual published figures. |
| "What's the HGBT scenario model based on?" | The model calculates revenue impact using HGBT price × gas volume × 65% sensitivity factor, with cascading P&L effects. In production, this connects to our full financial model with 40+ variables. |
| "Why is FCF declining faster than profit?" | Free cash flow reflects CAPEX acceleration ($353M budget for 2026) in addition to margin compression — a strategic investment choice. |

### Technology Questions

| Likely Question | Prepared Answer |
|----------------|-----------------|
| "Is the AI making things up?" | No — every AI response uses RAG (Retrieval-Augmented Generation). The AI first retrieves relevant data from our verified databases, then generates narrative grounded in that data with source citations. |
| "Which AI model do you use?" | Primary: Claude (Anthropic) for all natural language tasks. We have automatic failover to GPT-4. Specialized models include IndoBERT for Indonesian sentiment, GridFM for gas flow, and TerraMind for satellite analysis. |
| "How secure is this?" | RBAC role-based access control — Board, Director, VP, Analyst levels. All data encrypted in transit and at rest. Audit trail on every query. AWS infrastructure in ap-southeast-1 (Singapore). |
| "What if the AI is wrong?" | Every insight includes source citations and confidence levels. Critical alerts require human acknowledgment. The system augments decision-making — it does not automate it. |
| "How real-time is 'real-time'?" | SCADA data: 15-second intervals. Financial data: batch processing every 4 hours from SAP. Market data: real-time streaming. Sentiment: 15-minute polling cycles. |

### Strategic Questions

| Likely Question | Prepared Answer |
|----------------|-----------------|
| "How much did this cost to build?" | The platform leverages open-source foundations (Next.js, Kafka, PostgreSQL) with cloud-native architecture. Key costs are AI model inference (Claude API), cloud infrastructure (AWS), and development team. |
| "How long to deploy to production?" | Sprint 1-2 are complete (infrastructure + finance). Full platform deployment is a 10-sprint (20-week) roadmap with progressive feature rollout. |
| "Can competitors build this?" | The AI models are commodities — our moat is the integration with PGN's specific data streams (SCADA topology, SAP financial model, pipeline GIS data, and domain-specific prompt engineering). |
| "What's the ROI?" | Conservative estimate: $5M+ annual value from (1) reduced analyst headcount for report generation, (2) faster incident response reducing downtime, (3) optimized HGBT negotiation prep, (4) earlier risk detection preventing losses. |
